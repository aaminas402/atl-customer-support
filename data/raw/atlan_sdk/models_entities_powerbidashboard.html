    PowerBIDashboard¶

    Instance of a Power BI dashboard in Atlan. Dashboards are a single page, often called a canvas, that tell a story through visualization.

    Complete reference

    This is a complete reference for the PowerBIDashboard object in Atlan, showing every possible property and relationship that can exist for these objects. For an introduction, you probably want to start with:

    • Snippets small, atomic examples of single-step use cases.
    • Patterns walkthroughs of common multi-step implementation patterns.
